# RadialLines

 俗にいう「集中線」です。<br>
Fs TouchDrawCenterの後継になります。

# 機能

* AutoSeed<br> ONでフレームごとにseedの値が変わります。
* Seed<br> 乱数のSEEDです。
* Value<br> 線の数です。実際は×４の数になります。
* CenterPos<br> 集中線の中心です。
* Length<br> 周囲から集中線の中心への距離のパーセント
* LengthRand<br>Lengthの乱雑さになります。よく勘違いされますが０%で変化が無くなり100％で乱雑さが最大になり餡巣。
* Weight<br> 線の幅
* WeightRand<br> 線の幅の乱雑さ
* Opacity<br> 不透明度
* OpacityRand<br> 不透明度の乱雑さ
* Hyperbolid<br> アルファーチャンネルへの双曲線補完
* Noise<br> ノイズをかけます。
