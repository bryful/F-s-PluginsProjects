# Thin

 主線の細線化<br>
古のマルチペンチに同じ機能のやつありました。<br>
基本的に F's Thinをベタ移植しましたが、一応マルチスレッドで少し高速化（実感なし）

# 機能

* Inking Color Count<br>主線の色数
* Range<br>主線識別の許容値
* Target1-4<br>主線の色、最大８色
* ThinVale<br>削るピクセル数
* Ignore White<br>白では塗りつぶさない
* Ignore Transparrent<br>透明では塗りつぶさない
* Refine Edges<br>線のエッジ識別を強くする（悪影響あるかも）
