# SpeedLines

 平行な効果線です。<br>
 回転とか簡単にできます。


# 機能

* value<br>線の数
* position
  * length1_value<br>　右側の線の数。角度０の時の右側です
  * length1_Far<br>　右側の線の遠い方の範囲
  * length1_Near<br>　右側の線の近い方の範囲
  * length2_value<br>　左側の線の数
  * length2_Far<br>　左側の線の遠い方の範囲
  * length2_Near<br>　左側の線の近い方の範囲
  * center<br>　効果線の中心
  * autoSeed<br>　ＯＮで自動的に動きます
  * seed<br>　randomseedです。この値を動かせば適当に動きます。
* line
  * length<br>　線長さ
  * lengthRand<br> 線の長さのばらつき
  * size<br>　線の太さ
  * sizeRand<br> 線の太さのばらつき
  * opacity<br>　不透明度
  * opacityRand<br> 不透明度のばらつき
  * blue<br> ぼかす
  * noize<br> ノイズ。
* color
  * colorCount<br> 使う色数。
  * color1..8<br> 色。８色まで
* blend with orogonal<br> 元絵を合成します。
