# TargetGradRadical

 指定した範囲に球形グラデーションをかけるエフェクトです。<br>

## パラメータ
* target<br>targetColorsが指定した色、alphaOnは不透明部分、allは画像全体
* targetEnabled？<br>target?を有効にするかどうか
* target?<br>ターゲット。8色まで指定できます。
* invert<br>グラデーションの向きを反転。
* center<br>グラデーションの中心。
* radius<br>グラデーションの半径。
* feather<br>グラデーションのぼかし具合。
* hyperbolic<br>グラデーションのアルファーチャンネルを双曲線的に変化させます。
* aspect<br>球の左右比率を調整します。
* angle<br>球を回転させます。aspectが100以外のときに有効です。
* colorRable:Losd<br>色パラメータをロードします。
* colorRable:Save<br>色パラメータを保存します。
