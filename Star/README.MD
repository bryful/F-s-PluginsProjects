# Star

 クロスフィルターです。<br>
F's StarというよりF's Star_Colorfulの後継です。
マルチスレッドにしたので少し高速になってます。

# 機能

* Overall Angle<br>回転角度です。
* Overall Intensity<br>強さ
* Overall Length (px)<br>クロスの長さ
* hyperbolic<br>双曲線補完。
* Color Mode
  - Original 元の色で
  - CustomColor 
* CustomColor
* Threshold low<br>元との閾値の最低値（レベル補正の左側）
* Threshold High<br>元との閾値の最高値（レベル補正の右側）
* Line
  - Line 1 (Vertical)
    - AngleOffset_? Overall Angleとのオフセット
    - Length_?(%) 長さ。Overall Lengthとのスケール
    - Intensity_?(%) 強さ　Overall Intensityとのスケール
  - Line 2 (Diagonal 1)
  - Line 3 (Horizontal)
  - Line 4 (Diagonal 2)
