# Noise

 ノイズです。<br>
NF-Grainと比べて粒が小さいのが特徴です。中身はF's AnimatedNoiseとほぼ同じです。

# 機能

* Auto Seed<br> ONでSeedに自動的にフレーム数を足して勝手に動くようにします。
* Seed<br> ランダムの初期値です。
* Noise Amount<br> ノイズの数です。画面のピクセル総数のパーセントになります。
* Noise Intensity<br> ノイズの濃さ(%)です。実際はランダムなので最大値です。
* Accent Amount<br> アクセントのノイズの数。Noise Intensityに影響を受けます。
* Accent Intensity<br> アクセントノイズの濃さ（％）
* Color<br> カラーノイズにします。
