# ColorSampler

パスの頂点の座標の色を拾いUIに反映させます。<br>
<br>

SilverLinkの**MaskDropper.aex**のパクリエフェクトです。<br>
Kさん、すみません。<br>

# 機能

* Get Colors From Mask<br>パスマスクの頂点の位置の色を拾います。32点まで。
* Edit Color Names<br>色パラメータの名前を変更するダイアログを開きます。
