# InkingLineBlur

 主線をぼかして、指定した色のところだけ残します。

# 機能

* Inking Colors
  - Inking Color Count 主線の数
  - Inking? 主線の色。最大4色
  - Inking Color　Original or CustumColor
  - CustumColor
* MinMax<br> 最大最小
* Blur<br> ぼかし幅
* Hyperbolic<br> 双曲線補完
* Target Colors 
  - TargetMode 残す場所の数。Allだと不透明部分
  - Targer? ボケを残す場所の色
