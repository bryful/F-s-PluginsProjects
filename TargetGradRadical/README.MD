# TargetGrad

 指定した範囲にグラデーションをかけるエフェクトです。<br>

## パラメータ
* target<br>targetColorsが指定した色、alphaOnは不透明部分、allは画像全体
* targetEnabled？<br>target?を有効にするかどうか
* target?<br>ターゲット。8色まで指定できます。
* swapPoints<br>グラデーションの始点と終点を入れ替えます。
* hyperbolic<br>グラデーションのアルファーチャンネルを双曲線的に変化させます。
* 2Point:start/last<br>グラデーションの始点と終点を指定します。
* autoPos<br>始点と終点をターゲット範囲から自動的に設定します。
* rot<br>autoPosが有効なときに、始点と終点の角度を調整します。
* starPercent<br>始点の位置を調整します。
* lastPercent<br>終点の位置を調整します。
* offsetX<br>X位置をオフセットします。
* offsetY<br>Y位置をオフセットします。
* guideLines<br>グラデーションのガイドラインを表示します。
* guideColor<br>ガイドラインの色を指定します。
* colorRable:Losd<br>色パラメータをロードします。
* colorRable:Save<br>色パラメータを保存します。
