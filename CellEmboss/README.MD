# CellEmboss

選択した色にのみエンボスを付けるエフェクトです。<br>
F's CellGradの後継になります。<br>
<br>
適応させると透明になります。びっくりしないように。

# 機能

* colorCount <br> 有効なTargetColorの数です。AlphaOnだと不透明部分になります。<br>
* TargetColors <br> ターゲットになる色です。<br>
* direction <br> エンボスの方向です。
* mode 実行モードです。
  - Hi ハイライト方向のみ描画
  - Lo ハイライトの逆方向のみ描画
  - Hi-Reverce Hiをアルファー反転させたものです。
  - Hi+Lo HiとLoを両方表示
* Hi
  - distance_Hi Hiの幅
  - blue_Hi Hiのぼかし幅
  - color_Hi Hiの色
* Lo
  - distance_Lo Loの幅
  - blue_Lo Loのぼかし幅
  - color_Lo Loの色
