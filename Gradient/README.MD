# Gradient

グラデーションです。<br>
標準との違いは、

* 不透明度も指定できる
* RGB/Alpha個別に双曲線補完が加えられる

です。<br>
光の表現を目標にしたものです。


## 機能

* draw<br>機能切り替え。Linear（直線）/ Radial(球形)が選べます。
* startPoint<br> 開始点の位置
* startColor<br> 開始点の色
* startOpacity<br>開始点のの不透明度（％）
* endPoint<br> 終了点の位置
* endColor<br> 終了点の色
* endOpacity<br>終了点のの不透明度（％）
* swapColor<br>押すと開始点と終了点のいろ・不透明度を入れ替える。
