# MazzleFlash

　マズルフラッシュ。<br>
　BurstLinesのパラメータを変えただけのものです。そんなに用途はないと思いますが、面白そうだったので追加しました。

# 機能

* center<br>　位置
* count<br>　三角線の数。実際の数はcount*branchになります。
* rootSize<br>根元の太さ
* length<br>　線の長さ
* branch<br>
* direction<br> 回転。angleの方向。
* angle<br> 描画の範囲
* flash_scale<br> ONにするとscale/scale2ndをフレームごとに切り替える
* scale/scale2nd<br>　スケール
* sparks
  - sparkCount　細かい粒の数。実際はsparkCount*sparkSubCount個になる。
  - sparkSubCount 
  - sparkScale 粒の大きさ
* color<br> 色
* autoSeed<br>ONでフレームごとに描画
* seed<br>シード値
